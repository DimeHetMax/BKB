3. Visual Style

The visual style is modern, outdoors, friendly and professional. The website should use real photography of bikes, trails, repairs and accessories. Images should be bright, clear and consistent.

4. Layout Guidelines

Desktop

Maximum content width: 1200px. Use 3 or 4 columns for cards and horizontal navigation.

Tablet

Use 2 columns where needed. Keep medium spacing and clear section separation.

Mobile

Use a single-column layout, burger navigation and large readable buttons.
